Domain Tool What it covers for you Dashboard? Notes
CSPM — Azure posture
CSPM Prowler 400+ checks: VNet, Firewall, Storage, Key Vault, PostgreSQL, Redis, App GW, Entra ID, DDoS config HTML report only CLI output to JSON/HTML/CSV. No live UI. Best tool for Azure breadth.
CSPM ScoutSuite Multi-cloud, good Azure coverage. Generates self-contained HTML report per scan. HTML report only Good for periodic audit snapshots. Not continuous.
CSPM Defender for Cloud (you have it!) Azure-native posture: secure score, recommendations across all your resources Full dashboard You already have it — enable all plans. Free tier is limited but useful.
KSPM — Kubernetes posture
KSPM Kubescape AKS RBAC, pod security, network policies, CIS K8s benchmark, NSA hardening guide HTML + ARMO cloud UI Has a free cloud dashboard (ARMO). Best single KSPM tool for AKS.
KSPM kube-bench CIS K8s Benchmark checks against your AKS node config No dashboard CLI only, JSON/text output. Run once after AKS setup.
KSPM Polaris Pod spec checks: root containers, missing limits, missing probes, image tag :latest Web dashboard Deploy in-cluster. Has a built-in web UI accessible via port-forward. Lightweight.
KSPM kube-hunter Active hunting for attack paths in AKS: exposed APIs, priv escalation No dashboard Run periodically as a pen-test, not continuous. CLI report.
CWPP — Runtime protection
CWPP Falco Real-time syscall monitoring in AKS pods, Kafka VMs, App Service. Alerts on shell in container, suspicious outbound, /etc writes Falco UI (community) Deploy as DaemonSet. Most important runtime tool. Sends to webhook/Slack/Teams.
CWPP Tracee eBPF-based runtime, similar to Falco. Good for AKS newer kernels. No dashboard Aqua open-source. Use as Falco alternative, not alongside.
CIEM — Identity & entitlements
CIEM AzureHound + BloodHound Maps Entra ID, Azure RBAC, service principals, managed identities, attack paths to Key Vault / subscriptions BloodHound UI BloodHound CE (Community Edition) is free with full graph UI. Best tool for Entra ID risk.
CIEM Stormspotter Azure identity relationship graph: who can access what, lateral movement paths Neo4j web UI Microsoft research project. Requires Neo4j setup. More setup than BloodHound.
Container image scanning
Image sec Trivy Scan ACR images, Helm charts, K8s cluster, Dockerfiles for CVEs + misconfigs + secrets No dashboard CLI/JSON output. Integrate into Azure DevOps / GitHub Actions. Industry standard.
Image sec Grype + Syft CVE scanning + SBOM generation for container images No dashboard Pair together. Syft generates SBOM, Grype scans it for CVEs.
IaC security (shift-left)
IaC Checkov Scans Terraform, Bicep, ARM, Helm charts, K8s YAML, Dockerfile before deploy No dashboard Run in CI/CD pipeline. JSON/JUnit output. Plugin for VS Code too.
IaC KICS Same as Checkov, very broad IaC support, strong Terraform + K8s YAML coverage No dashboard Checkmarx open-source. Good alternative or supplement to Checkov.
Secrets detection
Secrets Gitleaks Scans git repos + history for Azure connection strings, storage keys, Entra client secrets No dashboard Run as pre-commit hook + CI pipeline gate. Fast, accurate.
Secrets TruffleHog Deep git history scan, finds secrets committed years ago and deleted No dashboard Slower than Gitleaks but catches more historical leaks. Run once on all repos.
SSPM — O365 posture
SSPM ScubaGear CISA-developed. Checks O365: MFA, Teams external access, Exchange DKIM/DMARC, SharePoint sharing HTML report only PowerShell tool by CISA (US govt). Free, authoritative O365 benchmark. Run monthly.
